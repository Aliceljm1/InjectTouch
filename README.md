# InjectTouch
模拟触控事件，支持输入触控点数据文件批量模拟.
demo文件:touchinfo.txt,
文本规则：LeftDown 100,100 0 0号笔迹左键在点100,100处按下，Delay 15 1 ，1号笔迹延时15ms,MoveTo 683,482 2, 2号笔迹移动到xx, LeftUp 200,200 3，3号笔迹左键在xx处弹起
详细参考按键精灵脚本文件
